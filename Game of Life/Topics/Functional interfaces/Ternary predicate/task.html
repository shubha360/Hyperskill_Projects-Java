Ternary predicate

You need to write your own functional interface called TernaryIntPredicate and use it with a lambda expression. The interface must have a single non-static (and non-default) method test with three int arguments and that returns a boolean value.

Besides, you need to write a lambda expression with three int arguments using your TernaryIntPredicate.

The lambda expression has to return true if all passed values are different, and, otherwise, false. The name of the instance is allValuesAreDifferentPredicate. It should be a static field.

There is a template to be used for your solution. Please, do not change it.


Sample Input:
1 1 1

Sample Output:
False


Sample Input:
2 3 4

Sample Output:
True
