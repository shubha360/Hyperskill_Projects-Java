Find a value's index

The best way to understand the nature of algorithms is to try writing one by yourself!

Implement int findIndex(int[] numbers, int value) method to search an index of an integer element value among non-negative integer numbers array. If numbers contains several value elements, return the index of the last one. In case there are no value elements in the numbers array, return -1.


Sample Input:
8 11 3 7 5 10 5
5

Sample Output:
6



Memory limit: 256 MB
Time limit: 8 seconds
