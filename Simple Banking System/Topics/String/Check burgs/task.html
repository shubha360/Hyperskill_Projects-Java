Check burgs

Write a program that reads the name of a city and checks if the name ends with "burg".

Keep in mind, a city can have a short name.

The program should output true or false.


Sample Input:
Augsburg

Sample Output:
true


Sample Input:
Saint Petersburg

Sample Output:
true


Sample Input:
Tu

Sample Output:
false



Memory limit: 256 MB
Time limit: 8 seconds
